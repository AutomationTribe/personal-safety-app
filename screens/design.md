# Design system

## Colours
- Primary: #1B4FD8 (deep blue — buttons, active states)
- Danger: #DC2626 (SOS button, errors)
- Success: #16A34A (confirmations)
- Background: #F8FAFC (app background)
- Surface: #FFFFFF (cards)
- Text primary: #0F172A
- Text secondary: #64748B
- Border: #E2E8F0

## Typography
- Font: Inter (Expo Google Fonts)
- Heading: 24px bold
- Subheading: 18px semibold
- Body: 16px regular
- Caption: 13px regular, text secondary colour

## Spacing
- Base unit: 8px
- Screen padding: 24px horizontal
- Card padding: 16px
- Gap between elements: 12px or 16px

## Components
- Primary button: full width, 52px height, primary colour, 
  12px border radius, white text 16px semibold
- Danger button: same as primary but danger colour
- Input field: 52px height, border colour border, 8px radius,
  16px padding horizontal, body text
- Card: surface colour, 12px radius, 16px padding, 
  1px border colour border

## Tone
Clean, serious, trustworthy. This is a safety app — no gradients, 
no decorative elements. Clarity over style.
